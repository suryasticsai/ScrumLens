📈 SprintAnalyzer v0.3

Automated Sprint Summary Generation with Excel Scripts & PivotTables

/**
 * Title: SprintAnalyzer v0.3
 * Purpose: Automates sprint data aggregation and PivotTable creation
 * Version: 0.3 | Ecosystem: SprintVerse01
 * Developer: Sai V (PL6K130)
 */
function main(workbook: ExcelScript.Workbook) {
  try {
    // Get active worksheet
    const selectedSheet = workbook.getActiveWorksheet();
    
    // Define range and check for overlapping tables
    let range = selectedSheet.getRange("A1:DT51");
    const existingTables = selectedSheet.getTables();
    
    // Create new table with auto-adjustment
    const newTable = workbook.addTable(range, true);
    
    // Perform text replacements for standardization
    const replacements = [
      { find: "[M1]-", replace: "[M1] - " },
      { find: "6101", replace: "INC" },
      { find: "Analysis ", replace: "Analysis completed for " }
    ];
    
    replacements.forEach(({ find, replace }) => {
      newTable.getRange().replaceAll(find, replace, { completeMatch: false });
    });
    
    // Create PivotTable on new worksheet
    let SprintAnalysisView = workbook.getWorksheet("SprintAnalysis_View");
    if (!SprintAnalysisView) {
      SprintAnalysisView = workbook.addWorksheet("SprintAnalysis_View");
    }
    
    const newPivotTable = workbook.addPivotTable("PivotTable1", newTable, SprintAnalysisView.getRange("A3"));
    newPivotTable.addFilterHierarchy(newPivotTable.getHierarchy("Custom field (Team)"));
    newPivotTable.addRowHierarchy(newPivotTable.getHierarchy("Component/s"));
    
  } catch (error) {
    console.log("Error:", error.message);
  }
}

let tagCounter = 0;

// Handle tag input
document.getElementById('tagInput').addEventListener('keydown', function(event) {
  if (event.key === 'Enter' && this.value.trim()) {
    addTag(this.value.trim());
    this.value = '';
  }
});

// Drag and drop functionality
function drop(event) {
  event.preventDefault();
  const data = event.dataTransfer.getData("text");
  const tag = document.getElementById(data);
  
  if (tag) {
    const clonedTag = tag.cloneNode(true);
    clonedTag.id = `tag-${tagCounter++}`;
    clonedTag.style.backgroundColor = getRandomColor();
    event.target.appendChild(clonedTag);
  }
}

// Generate JQL Query
function getTags() {
  const container = document.getElementById('tagsDropContainer');
  const tagNames = [];
  const tags = container.getElementsByClassName('tag');
  
  for (let tag of tags) {
    tagNames.push(tag.textContent.slice(0, -1).trim());
  }
  
  const jqlQuery = tagNames.includes('IN')
    ? `${tagNames.slice(0, tagNames.indexOf('IN')).join(' ')} IN (${tagNames.slice(tagNames.indexOf('IN') + 1).join(', ')})`
    : `issuekey in (${tagNames.join(', ')})`;
  
  document.getElementById('tagsOutput').value = jqlQuery;
}
🧙 SprintPlanWizard - Advanced Filtering

Intelligent Row Filtering with Real-time Progress Tracking

v0.6 • TypeScript 
/**
 * SprintPlanWizard - Advanced Sprint Data Filtering
 * Version: 0.6 | Ecosystem: SprintVerse01
 * Developer: Sai V (PL6K130)
 * Purpose: Filter colored cells and create dynamic PivotTables
 */

function main(workbook: ExcelScript.Workbook) {
  const now = new Date();
  let sheet = workbook.getActiveWorksheet();
  let usedRange = sheet.getUsedRange();
  
  let rowCount = usedRange.getRowCount();
  let colCount = usedRange.getColumnCount();
  
  // Create destination sheet
  let newSheet = workbook.addWorksheet("ColoredPivotTab");
  
  // Add progress loader
  let loaderCell = newSheet.getRange("A1");
  loaderCell.setValue("Processing: 0%");
  
  // Copy headers
  let headerRow = usedRange.getRow(0);
  newSheet.getRangeByIndexes(1, 0, 1, colCount).copyFrom(headerRow);
  
  // Scan and filter rows with colored cells
  let outRow = 2;
  for (let i = 1; i < rowCount; i++) {
    let rowRange = usedRange.getRow(i);
    let includeRow = false;
    
    // Check for colored cells
    for (let j = 0; j < colCount; j++) {
      let cell = rowRange.getCell(0, j);
      let fillColor = cell.getFormat().getFill().getColor();
      
      if (fillColor && fillColor !== "#FFFFFF") {
        includeRow = true;
        break;
      }
    }
    
    if (includeRow) {
      newSheet.getRangeByIndexes(outRow, 0, 1, colCount).copyFrom(rowRange);
      outRow++;
    }
    
    // Update progress
    let progress = Math.round((i / rowCount) * 100);
    loaderCell.setValue(`Processing: ${progress}%`);
  }
  
  // Create PivotTable
  let coloredRange = newSheet.getRangeByIndexes(1, 0, outRow - 1, colCount);
  let pivot = workbook.addPivotTable("ColoredPivot", coloredRange, newSheet.getRange("A3"));
  
  // Configure pivot fields
  pivot.addRowHierarchy(pivot.getHierarchy("Team"));
  pivot.addDataHierarchy(pivot.getHierarchy("Key"), ExcelScript.AggregationFunction.count);
  
  loaderCell.setValue("✅ 100% Complete");
}

📖 SprintAnalyzer v0.3 - Usage Manual

🎯 Overview

SprintAnalyzer automates the creation of sprint summary tables and PivotTables from your Excel data. It standardizes data formatting and creates dynamic analysis views.

📋 Prerequisites

  • Excel with Office Scripts enabled
  • Data in range A1:DT51 (customizable)
  • Column headers in the first row

🚀 How to Use

  1. Open your Excel workbook with sprint data
  2. Go to Automate → New Script
  3. Paste the SprintAnalyzer code
  4. Click Run to execute
  5. Check the new SprintAnalysis_View sheet for results

✨ Key Features

🔄
Auto-Adjustment

Detects overlapping tables and adjusts range automatically

📝
Text Standardization

Replaces prefixes and codes for consistent formatting

📊
Dynamic PivotTables

Creates configured pivot tables with team and component hierarchies

⚙️ Customization

To modify the range, update line 26:

let range = selectedSheet.getRange("A1:DT51");

To add more replacements, extend the replacements array with new find/replace pairs.

🐛 Troubleshooting

  • Script fails: Ensure data exists in the specified range
  • No PivotTable created: Check that required column headers exist
  • Overlapping error: Script automatically adjusts range below existing tables

📖 MSA Toolkit - JQL Query Builder Manual

🎯 Overview

MSA Toolkit is an interactive JQL (Jira Query Language) builder that simplifies complex query construction through drag-and-drop interface.

🎮 How to Use

  1. Create Tags: Type in the input field and press Enter
  2. Drag Operators: Drag default tags (IN, AND, OR, =, >, <, etc.) to the drop zone
  3. Build Query: Arrange tags in the correct order
  4. Generate JQL: Click "Get Tags" button
  5. Copy Result: Copy the generated JQL query

🏷️ Available Operators

  • IN: Checks if value is in a list
  • AND: Logical AND operator
  • OR: Logical OR operator
  • =, >, <, >=, <=: Comparison operators

💡 Examples

Example 1 - Simple Issue List:

issuekey in (PROJ-123, PROJ-124, PROJ-125)

Example 2 - Using IN Operator:

status IN (Open, In Progress, Ready)

Example 3 - Complex Query:

project = PROJ AND status IN (Open, In Progress) AND assignee = currentUser()

📖 SprintPlanWizard - Usage Manual

🎯 Overview

SprintPlanWizard filters Excel data based on cell colors and creates intelligent PivotTables with real-time progress tracking.

📋 Prerequisites

  • Excel with Office Scripts enabled
  • Data with colored cells (highlighting important rows)
  • Column headers in the first row
  • Optional: "Story Point" column for zero-value filtering

🚀 How to Use

  1. Highlight important rows in your sprint data with colors
  2. Open Excel and go to Automate → New Script
  3. Paste the SprintPlanWizard code
  4. Click Run to execute
  5. Monitor progress in cell A1 (shows percentage)
  6. View results in the new ColoredPivotTab sheet

🔍 Filtering Logic

The script filters rows that meet ANY of these criteria:

  • Contains colored cells (any color except white)
  • Has zero or empty value in "Story Point" column

✨ Key Features

📊
Smart Filtering

Automatically detects colored cells and zero-value rows

⏱️
Real-time Progress

Shows processing percentage in real-time

📈
Dynamic Pivot

Creates configured PivotTable with Team and Key hierarchies

⚙️ Customization

To change the new sheet name, modify line 38:

let newSheetName = "ColoredPivotTab";

To add more pivot hierarchies, extend the configuration after line 104.

🐛 Troubleshooting

  • Progress stuck: Check data range size; very large datasets may take time
  • No filtered results: Ensure rows have colored cells or zero Story Points
  • PivotTable empty: Verify Team and Key columns exist in your data
