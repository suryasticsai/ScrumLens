  ABOUT:
  ------
  Chaos2Clarity (C2C) is the core philosophy behind ScrumLens — turning 
  messy, unstructured team conversations into clean, actionable Scrum insights.

  Paste raw Teams / Slack / Discord transcripts. The tool auto-detects roles,
  tags decisions/actions/risks, builds a visual message chain, and calculates
  sprint health — all client-side, zero backend, zero privacy risk.
